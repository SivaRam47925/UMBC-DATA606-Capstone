## Project Summary

This project focuses on predicting ICU patient mortality using machine learning models trained on clinical healthcare data.

The final deployed system includes:

- Gradient Boosting predictive model
- Interactive Streamlit dashboard
- Feature importance analysis
- EDA visualizations
- Real-time mortality risk prediction

# Notebooks Folder

This folder contains Jupyter notebooks used for exploratory data analysis, preprocessing, feature selection, and machine learning modeling for the ICU Mortality Prediction project.

---

## Files

### `EDA.ipynb`

Contains:

- Data exploration
- Missing value analysis
- Feature distributions
- Correlation analysis
- Visualizations and insights

---

### `Modeling.ipynb`

Contains:

- Data preprocessing
- Feature selection
- Machine learning training
- Model evaluation
- ROC-AUC analysis
- Feature importance analysis

---

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Plotly
- Matplotlib

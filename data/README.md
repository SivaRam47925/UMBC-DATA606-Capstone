# Data Folder

This folder contains datasets used for the ICU Mortality Prediction project.

## Files

- `training_sample.csv` - Sample ICU dataset used for exploratory data analysis and machine learning modeling.

## Dataset Source

WiDS Datathon 2020 ICU Mortality Prediction Dataset:

https://www.kaggle.com/competitions/widsdatathon2020/data

## Notes

The original dataset was reduced to a smaller representative sample due to GitHub file size limitations.
